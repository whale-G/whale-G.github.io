JDK（Java Development Kit）是 Java 开发的核心工具包，包含编译、运行和调试 Java 程序所需的所有工具和库。它主要由 JVM（Java 虚拟机）、JRE（Java 运行时环境）和 Java 核心类库组成。

JVM 是跨平台运行的基础，负责字节码的加载、执行和内存管理；JRE 提供运行 Java 应用的环境；核心类库则提供了丰富的 API 支持。
